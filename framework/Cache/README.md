# Пакет компонентов Ge\Cache входит в состав Ge Framework.

Пакет компонентов предоставляет общую систему кеширования для PHP. Может кэшировать различные шаблоны (класс, объект, вывод и т.д.) используя разные адаптеры хранения (DB, File, Memcache и т.д.).

## Ресурсы
- [Ge Framework](https://rosgear.ru/framework/)
