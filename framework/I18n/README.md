# Пакет компонента Ge\I18n входит в состав Ge Framework.

Ge\I18n поставляется с полным пакетом переводов, который поддерживает все основные форматы и включает популярные функции, такие как plural. Компонент Translator в основном свободен от зависимостей, за исключением возврата к языку по умолчанию, где он полагается на расширение Intl PHP.

Сам транслятор инициализируется без каких-либо параметров, так как его настройка необязательна. Транслятор без каких-либо переводов фактически ничего не сделает, а просто вернет заданные идентификаторы сообщений.

## Ресурсы
- [Ge Framework](https://rosgear.ru/framework/)
