# Пакет компонента Ge\Log входит в состав Ge Framework.

Пакет компонента Ge\Log предназначен для ведения журнала общего назначения. Он поддерживает несколько бэкэндов журналов, форматирование сообщений, отправляемых в журнал, и фильтрацию сообщений при записи вы журнал.

## Ресурсы
- [Ge Framework](https://rosgear.ru/framework/)
