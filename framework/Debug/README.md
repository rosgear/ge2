# Пакет компонента Ge\Debug входит в состав Ge Framework.

Компонент, помогает отлаживать приложения PHP. В частности, он предлагает статический метод Ge\Debug\Debug::dump(), который печатает или возвращает информацию о выражении. Этот простой метод отладки является обычным, поскольку его легко использовать в произвольном порядке и не требуется инициализация, специальные инструменты или среда отладки.

## Ресурсы
- [Ge Framework](https://rosgear.ru/framework/)
