# Пакет компонента Ge\Db входит в состав Ge Framework.

Пакеты компонента абстрагирует доступ к базе данных с помощью объектно-ориентированного API для построения запросов. Компонент использует разные адаптеры хранения для доступа к различным поставщикам баз данных, таким как MySQL, PostgreSQL, Oracle, IBM DB2, Сервер Microsoft Sql, PDO и т.д.

## Ресурсы
- [Ge Framework](https://rosgear.ru/framework/)
